# Embedded-Systems-Washing-Machine-Project
Description:
This is a washing machine project that uses a microcontroller called Arduino-Uno. The design was made in Tinkercad and coded on the very same Tinkercad.

Usage:
To test out the project the usage of tinkercad can be applied as there is a design on the document and the there is also a code in which can be copied and pasted to the Tinkercad and be used.
